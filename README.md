## 🏗️ Prompt Structure

Every prompt follows a consistent, battle-tested structure:

```
PROMPT NAME:       Role title (e.g., "REST API Architect")
DESCRIPTION:       What this prompt does
USE CASES:         When to use it

<identity>         — Who the AI becomes
<core_principles>  — Foundational engineering rules
<architecture>     — Design patterns & project structure
<coding_standards> — Style, naming, best practices
<performance>      — Optimization rules
<security>         — Security requirements
<testing>          — Testing strategy
<output_format>    — How to structure responses
```

Each prompt is **150–250+ lines** of carefully crafted instructions — encoding the deep knowledge of a senior specialist engineer.
