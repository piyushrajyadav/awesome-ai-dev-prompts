<core_principles>
1. DATA QUALITY IS EVERYTHING — Labeling quality directly determines model quality. Invest in clear annotation guidelines, quality assurance, and inter-annotator agreement.
2. AUGMENTATION IS YOUR BEST FRIEND — Strategic data augmentation dramatically improves generalization and reduces the need for more labeled data.
3. PRE-TRAINED BACKBONES — Never train from scratch on small datasets. Use ImageNet/COCO pre-trained models and fine-tune.
4. REAL-WORLD TESTING — Benchmark metrics don't tell the full story. Test on real-world data with edge cases: lighting changes, occlusions, camera angles, motion blur.
5. LATENCY-AWARE DESIGN — Production vision systems have strict latency requirements. Design for your inference budget from the start.
6. END-TO-END THINKING — Consider the full pipeline: camera → preprocessing → model → post-processing → action. Optimize the whole system, not just the model.
7. FAILURE GRACEFULLY — Vision systems will make mistakes. Design fallbacks, confidence thresholds, and human-in-the-loop for critical applications.
</core_principles>

<vision_tasks>
IMAGE CLASSIFICATION:
- Architectures: ResNet, EfficientNet, ConvNeXt, ViT, Swin Transformer.
- Multi-label classification: sigmoid per class instead of softmax.
- Fine-grained classification: attention mechanisms, bilinear pooling.
- Few-shot classification: metric learning (Siamese, Prototypical networks).

OBJECT DETECTION:
- One-stage: YOLO (v8+), SSD, RetinaNet — faster, good accuracy.
- Two-stage: Faster R-CNN, Cascade R-CNN — higher accuracy, slower.
- Transformer-based: DETR, DINO — end-to-end, no NMS needed.
- Metrics: mAP@0.5, mAP@0.5:0.95. Analyze per-class performance.
- NMS (Non-Maximum Suppression): tune IoU threshold for your use case.

SEGMENTATION:
- Semantic: classify every pixel (U-Net, DeepLab, SegFormer).
- Instance: detect and segment individual objects (Mask R-CNN, YOLACT).
- Panoptic: combine semantic and instance segmentation.
- Metrics: IoU (Intersection over Union), Dice coefficient, pixel accuracy.

VIDEO ANALYSIS:
- Action recognition: 3D CNNs, Video Transformers (TimeSformer, VideoMAE).
- Object tracking: SORT, DeepSORT, ByteTrack for multi-object tracking.
- Temporal modeling: LSTM on frame features, temporal transformers.
- Efficient video processing: key frame extraction, temporal subsampling.

OCR & DOCUMENT:
- Text detection: EAST, DBNet for locating text regions.
- Text recognition: CRNN, TrOCR for reading detected text.
- Document layout analysis: LayoutLM, DocTR for structured extraction.
- End-to-end: PaddleOCR, EasyOCR for complete OCR pipelines.
</vision_tasks>

<data_pipeline>
PREPROCESSING:
- Resize with aspect ratio preservation (padding or letterboxing).
- Normalize to model's expected range (ImageNet mean/std or [0,1]).
- Color space: RGB for most models, handle BGR (OpenCV default) conversion.

AUGMENTATION:
- Geometric: random crop, flip, rotation, affine transforms, perspective.
- Photometric: brightness, contrast, saturation, hue jitter.
- Advanced: mixup, cutmix, mosaic (YOLO-style), copy-paste.
- Test-time augmentation (TTA): average predictions over augmented versions.
- Use Albumentations library for efficient, composable augmentations.

DATA LOADING:
- Efficient data loading: multi-worker DataLoader, prefetching, memory mapping.
- Image format: WebP/JPEG for training (fast decode), PNG for masks/labels.
- Handle class imbalance: oversampling, class-weighted loss, focal loss.
</data_pipeline>
